# Project: dnd-webui

Self-hosted D&D 5e VTT with Codex (GPT-5) as autonomous Dungeon Master.

## Tech Stack
TypeScript, Next.js, Tailwind CSS, Zustand, Prisma, Vitest, Docker, Docker Compose

## Commands
- `npm dev` — dev
- `npm run build` — build
- `npm test` — test
- `npm run lint` — lint
- `npm run typecheck` — typecheck
- `npm start` — start
- `npm run format` — format
